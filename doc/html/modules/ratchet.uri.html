Module ratchet.uri

The uri library provides a URI system for implementation of connection strings. Each schema registered in an object of this library will simply call a handler function, no logic is inherent.

Functions

__call (self, what)
Meta-method allowing a uri object to be called directly to resolve URI strings. This method is identical to resolve().

Parameters

  • self: the uri object.
  • what: the URI to resolve.
new (default_schema)
Returns a new uri object.

Parameters

  • default_schema: optional schema string to use when none is given.

Return value:

a new uri object.
register (self, schema, func)
Registers a new schema and handler. As per RFC 2396, a schema is recognized by the segment before the first ':' in the URI string. The handler function is then called with the remaining, schema-specific portion.

Parameters

  • self: the uri object.
  • schema: the schema string.
  • func: the handler function.
resolve (self, what)
Resolves the given URI based on its schema prefix or according to the default schema. The return values are passed directly from the schema handler function.

Parameters

  • self: the uri object.
  • what: the URI to resolve.
